# FitnessBodyRecomposition 🏋️‍♂️💪

A modern Android application designed to help users track their fitness journey, nutrition, and body recomposition progress. Built with Jetpack Compose and following modern Android development best practices.

## 🚀 Features

- **Dashboard**: A comprehensive overview of your daily activity, calories, and goals.
- **Nutrition Tracking**: Log and monitor your daily food intake and macro-nutrients.
- **Progress Tracking**: Track your weight, body measurements, and visualize your progress over time.
- **Onboarding**: A seamless experience to set up your fitness profile and goals.
- **Modern UI**: Clean, futuristic design with a dark theme and accent colors.

## 🛠 Tech Stack

- **UI**: [Jetpack Compose](https://developer.android.com/jetpack/compose) for a declarative and modern UI.
- **Architecture**: MVVM (Model-View-ViewModel) with Navigation Compose.
- **DI**: [Koin](https://insert-koin.io/) for lightweight dependency injection.
- **Styling**: Material 3 with custom colors and Google Fonts (Inter).
- **Language**: [Kotlin](https://kotlinlang.org/) for concise and safe code.
